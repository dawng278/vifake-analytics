# Legacy Scripts

Thư mục này chứa các script khởi động cũ, giữ lại để tham chiếu lịch sử.

- `start_vifake.sh`: script bootstrap đời cũ (trỏ về hạ tầng cũ).

## Khuyến nghị hiện tại

- Dùng `make demo` hoặc `docker compose up --build -d` từ root repo.
- Không dùng script legacy cho luồng demo/chấm chính thức nếu không cần.
